## 개발 목적
이번 개발의 목적은 `agent-team`을 운영 명령형 도구 모음에서 한 단계 더 나아가, 사용자가 `bun atcli.js` 또는 별도 명령으로 실행했을 때 **즉시 자연어 입력 대기 상태로 진입하는 프로젝트 빌더 전용 TUI CLI 앱**으로 제공하는 것이다.

특히 이번 작업은 아래를 우선한다.

- `bun atcli.js` 기반 단일 실행 파일 진입점 제공
- 별도 확인/대체 경로로 `agent-team app` 또는 동등한 명령 제공
- Codex 스타일의 자연어 입력 대기 화면 제공
- 첫 자연어 goal 입력 후 `software-factory` 팀 자동 bootstrap 연동
- 같은 화면 안에서 진행 상태/agent 상태/activity/log를 확인할 수 있는 기본 UX 제공

## 개발 범위
1. `atcli.js` 단일 실행 진입점 추가
2. `agent-team app` 대체 실행 명령 추가
3. 자연어 입력 대기형 Ink TUI 앱 추가
4. 첫 입력을 `run "<goal>"` bootstrap으로 연결
5. 실행 후 동일 화면에서 상태 모니터링과 기본 follow-up 입력 경로 제공
6. 관련 테스트, README/예시/인수인계 문서 갱신

## 제외 범위
- direct OpenAI API 또는 기타 vendor direct API 연동
- 완전 자율 recursive agent generation 고도화
- 고급 slash command 시스템 전부 구현
- 원본 Codex CLI UI exact parity
- 원격 협업/멀티머신 transport 확장

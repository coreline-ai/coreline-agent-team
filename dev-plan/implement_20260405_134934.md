## 문서 기반 제약 사항
- `agent-team`의 표준 LLM/runtime 경로는 `Codex CLI`이며, direct API 경로 추가는 금지한다.
- upstream Agent Teams 관찰은 그대로 복사하지 않고, `직접 적용 / 부분 적용 / 참고 전용`으로 다시 판정한다.
- 현재 프로젝트의 핵심 강점은 `resume` / `reopen` / detached background worker / file-backed storage 이지만, 동시에 이 영역이 가장 깨지기 쉬운 안정성 축이다.
- 이번 workstream은 UI polish보다 **구조적 제한 명문화 + 운영 복구 정합성 + 비용 가드레일**을 우선한다.

| 비용 제한 | 포함 | 팀 규모 3~5명 가이드, broadcast 비용, 수확 체감 |
